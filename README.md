# nd-position

> 简单实用的定位工具，将一个 DOM 节点相对于另一个 DOM 节点进行定位操作。fork 自 [arale-position](https://github.com/aralejs/position)

## 安装

```bash
$ npm install nd-position --save
```

## 使用

```js
var Position = require('nd-position');
// use Position
```
